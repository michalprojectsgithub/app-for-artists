How it works

Desktop app → Phone / Browser → Local library

No account, no uploads. Everything stays on your machine, including phone transfers over the same Wi-Fi.

Chrome extension

Save images from any website straight into Quiet Room as references. The extension sends the image directly to your local library, so everything stays on your machine.

Download the extension

Grab the ZIP, then load it once in Chrome using Developer mode.

How to use it

Save a single image

Right-click an image and choose Save to References.

Find multiple references

Right-click anywhere and select Find references on the page.

Install in developer mode

1

Unzip the downloaded extension folder on your computer.

2

Open chrome://extensions in Chrome.

3

Enable Developer mode in the top-right corner.

4

Click Load unpacked and select the unzipped folder.
